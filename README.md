# ParametrizedMazeGen

A tool to generate parametrized maze.

## Version

Python 3.8.6

## Create virtualenv

```bash
python3 -m venv maze-env
```

## Activate virtualenv

```bash
source maze-env/bin/activate
```

## Install requirements

```bash
pip3 install -r requirements.txt
```

## Generate

```bash
python src/generate.py
```
